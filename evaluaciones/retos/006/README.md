# Reto 006

## pyIris

Iris es la [propuesta de @javierLianoRioz](https://github.com/JavierLianoRioz/Iris) que consiste en un redirector de mensajes estructurados, el cual captura todos los envíos de un emisor, los resume y envía este resumen a los clientes que se han suscrito.

Los mensajes pertenecen a una categoría.

## Reto

Proponga las estructuras de datos que utilizaría para desarrollar esta solución.

## Entregable

1. **Diagramas**: Representación visual UML de qué estructuras se utilizan y cómo se conectan
1. **Justificación**: Para cada operación, explicar **qué** estructura soporta esa operación y **por qué**.
1. **Compromisos**: Identificar de modo claro qué operaciones se sacrifican (si alguna) y el coste aceptado
1. **Casos límite**: Describa cómo el diseño que propone maneja:
    - Asignaturas sin alumnos suscritos
    - Alumnos sin asignatura inscrita
    - Exalumnos
    - Mensajes que no provienen de una asignatura
    - Mensajes no enviados
